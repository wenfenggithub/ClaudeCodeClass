## 看多论点："可复用性垄断"

### 核心论点陈述
做多SpaceX — 唯一具备规模化轨道级助推器复用能力的公司，拥有自我造血的卫星宽带现金引擎（星链：$114亿营收，63% EBITDA利润率），以及将本已巨大的成本优势进一步扩大至竞争对手10–20倍的下一代发射系统（星舰）。$1.75万亿估值虽然激进，但如果星舰成功且星链达到4,000万+用户，则具备合理性。

### 关键支柱

|---|---|---|---|
| P1 | **可复用性护城河扩大** | 猎鹰9复用 → 星舰快速复用。每公斤成本从~$2,700降至~$200（2028年）。无竞争对手展示可比复用能力。 | 星舰V3飞行节奏；在轨加注演示；竞争对手复用公告 |
| P2 | **星链增长轨迹** | 1,030万用户 → 2028年4,000万+。营收$114亿 → $300亿+。EBITDA利润率维持60%+。海事/航空/国防垂直领域加速。 | 季度用户增长；ARPU走势；Kuiper用户吸引力；流失数据 |
| P3 | **垂直整合优势** | 85-90%内部制造，全栈利润获取。无供应商瓶颈。竞争对手依赖第三方发动机/组件。 | 内部生产率；同行的供应商依赖程度 |
| P4 | **政府需求长期顺风** | NSSL Phase 3（$137亿总额）、SDA扩散型LEO、阿尔忒弥斯、金穹——均偏好低成本、高频次供应商。SpaceX是现任颠覆者。 | 新合同授予；预算拨款；中国反卫星试验推升紧迫性 |
| P5 | **IPO资金加速领先** | $750亿融资@$1.75万亿+ → 加速星舰/星链V3基础设施。竞争对手合计市值<$750亿。 | IPO定价；募集资金用途；资本支出效率 |

### 看多论点关键风险
- 星舰V3未能展示快速复用 → 估值逻辑崩溃
- Kuiper达到商业规模（1,000+颗卫星）→ 星链面临定价压力
- 马斯克治理：85.1%投票控制权，xAI/Tesla/Twitter分散注意力风险
- xAI现金消耗（2025年亏损$64亿）侵蚀星链利润 → 多年无自由现金流

### 目标价（看多逻辑兑现情况下）
- **2028E：** $2.5–3.0万亿估值（~15× $2,000亿EBITDA），条件：4,000万星链用户 + 星舰运行 + xAI盈亏平衡

### 止损触发（做多仓位）
1. 星舰V3发生灾难性故障或项目暂停
2. 星链用户增长停滞在1,500万以下（对比4,000万牛市情景）
3. 马斯克离开SpaceX或投票控制权被成功挑战
4. Kuiper达到500万+活跃用户并具备竞争力定价
5. Anthropic取消xAI算力合同（90天条款）

## 看空论点："万亿级执行风险"

### 核心论点陈述
做空SpaceX — $1.75万亿IPO估值（~94× 2025年P/S）同时计入了星舰完美执行、4,000万+星链用户和轨道AI算力的预期。公司2025年亏损$49.4亿。xAI每$1营收烧$2。马斯克持有85.1%投票控制权。Kuiper、中子号及中国国资背景竞争者正在入场。$1.75万亿的风险/回报呈现深度不对称的下行风险。

### 关键支柱

| # | 支柱 | 原始预期 | 跟踪指标 |
|---|---|---|---|
| P1 | **估值极端** | $1.75万亿 = ~94× 2025年P/S。Damodaran DCF估值：~$1.22万亿（~低于30%）。最乐观假设已被定价。无安全边际。 | IPO后交易；卖方首评覆盖；季度业绩vs指引 |
| P2 | **星舰执行风险未充分定价** | >$150亿已投入，在轨加注未经验证。V3近乎全新设计。任何重大延迟都将瓦解IPO逻辑。 | 星舰试飞结果；V3星链部署日期（目标2026年下半年） |
| P3 | **星链ARPU结构性下降** | ARPU：$99（2023）→ $66（Q1 2026）。主动"以价换量"策略，但如果量不能抵消，单位经济恶化。Kuiper入市加速ARPU下降。 | 季度ARPU；用户增长vs ARPU权衡；Kuiper定价公告 |
| P4 | **xAI是数十亿美元的锚** | 2025年亏损$64亿（对应$32亿营收）。Anthropic合同 = $12.5亿/月 — 可取消，提前90天通知。AI资本支出周期正在转向。 | xAI独立营收/利润；Anthropic合同续签；资本支出轨迹 |
| P5 | **治理与关键人集中风险** | IPO后85.1%投票控制权。马斯克同时运营Tesla、xAI、Twitter/X、DOGE。分散注意力和集中风险在$1.75万亿级别公司史无前例。 | 马斯克时间分配；股东诉讼；IPO后董事会独立性 |

### 看空论点关键风险
- 星舰按期成功 → 每公斤成本再降5–10倍 → 护城河无懈可击
- 星链达到逃逸速度（网络效应全球锁定用户）
- Kuiper/中子号/中国未能交付 → SpaceX保持70%+发射份额
- 在IPO动能和马斯克资本光环下做空时机危险

### 目标价（看空逻辑兑现情况下）
- **2028E：** $5000–8000亿（~4–5× 2028E EBITDA），条件：星舰推迟、星链1,500万用户（而非4,000万）、xAI重组、宽带竞争性定价

### 止损触发（做空仓位）
1. 星舰V3完成10+次成功快速复用飞行
2. 星链突破2,000万用户且ARPU稳定或上升
3. IPO后市值跌至$1.2万亿以下（估值趋于合理）
4. xAI签署多年期Anthropic续约合同且条款改善
5. 蓝色起源年发射节奏低于5次（竞争威胁消退）

## 速查：晨会一页纸摘要

**SPCX（Pre-IPO）— 多空双面分析**

- **公司概要：** SpaceX = 全球主导发射供应商（>80%份额）+ 全球最大卫星宽带运营商（1,030万用户）+ 一个亏损数十亿美元的AI业务（xAI）。2025年营收$187亿，净亏损$49.4亿。
- **投资背景：** ~3周后IPO，估值$1.75–2.0万亿。全部估值建立于星舰快速复用 + 星链4,000万+用户 + 轨道AI算力 — 三者需同时成立。
- **牛市情景（30%概率）：** 星舰成功 → 每公斤成本降10× → 星链V3部署 → 4,000万用户 → 2028年EBITDA $2,000亿 → $2.5–3.0万亿估值。
- **熊市情景（20%概率）：** 星舰推迟 → 星链困于猎鹰9 → Kuiper入市 → ARPU战 → xAI崩溃 → 治理危机 → $5,000–8,000亿估值。
- **基础情景（50%概率）：** 星舰部分成功，2,000–2,500万星链用户，Kuiper竞争，xAI拖累——大致持平IPO估值$1.5–2.0万亿。
- **核心问题：** 能否在不要求完美的价位买入？如IPO定价$1.5万亿以下，风险/回报显著改善。$2万亿+则适合做空。
- **决策触发：** IPO后交易区间 + Q2 2026财报（上市后首份）+ Q4 2026前星舰V3部署节奏。
