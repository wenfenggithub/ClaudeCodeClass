## 一、调研概述

### 1.1 调研目标

全面梳理中国大陆投资者目前可以购买的、被动跟踪标普500指数和纳斯达克100指数的全部公募基金产品，包括：

- **场内ETF**（上海/深圳证券交易所上市交易的QDII-ETF）
- **ETF联接基金**（场外申赎、以ETF为主要投资标的）
- **QDII指数基金**（场外直接申赎的QDII指数产品）
- **QDII-LOF**（深交所上市型开放式基金）
- **QDII-FOF**（以海外指数基金为投资标的的FOF产品）
- **等权重/策略指数**（跟踪标普500等权重等变体指数）

### 1.2 数据来源与核实方法

| 数据维度 | 来源1（主） | 来源2（交叉验证） | 核实状态 |
|---|---|---|---|
| 基金代码与名称 | 天天基金网 `fundgz` 实时API | `USA.xlsx` 全部份额Sheet | ✅ 逐一核验 |

| 申购/赎回状态 | 天天基金网 | 基金公司官网 | ⚠️ 以基金公司最新公告为准 |

## 二、标普500 vs 纳斯达克100 指数对比

### 2.1 指数基本信息

| 维度 | 标普500（S&P 500） | 纳斯达克100（NASDAQ 100） |
|---|---|---|
| **指数代码** | SPX / ^GSPC | NDX / ^NDX |
| **发布机构** | 标普道琼斯指数公司 | 纳斯达克交易所 |
| **成分股数量** | ~503（实际） | ~101（实际） |
| **选股范围** | 美国大盘股（市值加权） | 在纳斯达克上市的100家最大非金融公司 |
| **加权方式** | 自由流通市值加权 | 修正市值加权 |
| **再平衡频率** | 每季度 | 每季度（12月为年度再平衡） |
| **基准日期** | 1957年3月4日 | 1985年1月31日 |

### 2.2 行业分布对比

| 行业 | 标普500权重 | 纳斯达克100权重 |
|---|---|---|
| **信息技术** | ~31% | ~58% |
| **通信服务** | ~9% | ~15% |
| **可选消费** | ~10% | ~14% |
| **医疗保健** | ~12% | ~4% |
| **金融** | ~13% | ~0%（排除金融股） |
| **工业** | ~8% | ~3% |
| **能源** | ~4% | ~0.5% |
| **其他** | ~13% | ~5.5% |
| **科技+通信+消费合计** | **~50%** | **~87%** |

### 2.3 关键风险指标对比

| 指标 | 标普500 | 纳斯达克100 |
|---|---|---|
| **科技股集中度** | 中等（Top 10占~34%） | 极高（Top 10占~55%） |
| **历史最大回撤** | -56.8%（2008金融危机） | -81.1%（2000互联网泡沫） |
| **近10年年化波动率** | ~15-17% | ~20-24% |
| **近10年年化收益** | ~13-14% | ~18-20% |
| **夏普比率** | ~0.85 | ~0.90 |
| **股息率** | ~1.3% | ~0.7% |
| **利率敏感性** | 中等 | 高（成长股久期更长） |
| **汇率风险** | 人民币汇率波动影响 | 人民币汇率波动影响（更高波动放大效应） |

### 2.4 投资者适配场景

| 投资者类型 | 推荐指数 | 原因 |
|---|---|---|
| 保守型长期配置 | **标普500** | 行业分散、波动较低、股息更高、历史最大回撤更小 |
| 进取型成长投资 | **纳斯达克100** | 科技成长弹性大、长期收益更高、适合接受高波动的投资者 |
| 定投新手 | **标普500** | 波动适中、不会因巨大回撤恐慌赎回 |
| 卫星配置 | **纳斯达克100** | 作为核心-卫星策略的卫星部分，比例不超过总资产30% |
| 最大回撤敏感 | **标普500** | 历史上回撤恢复时间明显短于纳斯达克100 |
