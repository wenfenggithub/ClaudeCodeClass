## 二、子问题拆解

### 2.1 标普500指数基金的定义边界

| 子问题 | 判断标准 |
|---|---|
| 什么是"被动指数型"？ | 基金合同明确约定跟踪某一指数，基金经理无选股裁量权 |
| 什么是"场外指数基金"？ | 不通过证券交易所交易，通过基金公司、银行、第三方平台申购赎回 |
| 什么是"场内ETF"？ | 在沪深交易所上市的QDII-ETF产品 |
| 什么是"ETF联接基金"？ | 以ETF为主要投资标的的场外基金，实现间接跟踪指数 |
| 什么是"QDII-FOF"？ | 以海外指数基金（ETF）为投资标的的FOF产品，存在双重收费 |
| 如何区分"跟踪标普500"与"标普500等权重"？ | 业绩基准是否为 S&P 500 Index（市值加权）vs S&P 500 Equal Weight Index（等权重） |

### 2.2 纳斯达克100指数基金的定义边界

| 子问题 | 判断标准 |
|---|---|
| 纳斯达克100 vs 纳斯达克综合指数 | NDX跟踪100只最大非金融股，IXIC跟踪3000+只股票 |
| 纳斯达克100 ETF的识别 | 业绩基准明确标注"纳斯达克100指数"或其变体 |
| 是否排除科技主题基金？ | 是——主动管理型科技主题基金不纳入被动指数基金统计 |

### 2.3 产品筛选子问题

| 子问题 | 研究方式 |
|---|---|
| 哪些基金公司发行了标普500/纳指100 QDII产品？ | USA.xlsx 全部份额Sheet 提供完整清单 → fundgz API 逐代码核验 |
| 各基金的真实跟踪指数是什么？ | F10页面 + USA.xlsx 交叉确认 |
| A类/C类/D类/E类份额如何区分？ | A类前端收费，C类收销售服务费，D/E类为互联网平台专属份额 |
| 是否有美元份额？ | 多只产品有美元现汇份额（013425/013499/017642/017643/003718/012861等） |
| 哪些产品初版遗漏？ | 通过USA.xlsx全部份额Sheet对比发现7大类遗漏 |

### 2.4 数据核验子问题

| 子问题 | 核验方法 |
|---|---|
| 基金代码是否正确？ | 天天基金网 real-time API (`fundgz.1234567.com.cn/js/{code}.js`) 逐个验证 |
| 空返回代码是否存续？ | fundgz 返回 `jsonpgz()` 的代码，通过F10页面标题确认基金存续 |
| 费率数据是否准确？ | 以F10基本概况页为准，USA.xlsx运作费率列为交叉验证 |
| 规模数据是否最新？ | USA.xlsx 标注为2026-03-31季报数据，时效性可接受 |
| USA.xlsx数据是否可靠？ | 通过API核验全部代码存在性，确认USA.xlsx数据准确 |
