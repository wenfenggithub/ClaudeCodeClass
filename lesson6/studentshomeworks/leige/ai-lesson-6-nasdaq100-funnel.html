1.1 什么是漏斗分析？

FUNNEL ANALYSIS 101

漏斗分析（Funnel Analysis）是一种用于追踪用户从初始接触到最终转化全过程的数据分析模型。它帮助分析师定位每个环节的流失点，从而优化转化率。

核心思维：逐层筛选、逐级收敛——从海量数据中逐步聚焦到最有价值的目标。

3.3 核心估值指标三维评估

VALUATION METRICS

📊 PE（市盈率）

股价 ÷ 每股收益。
反映市场对公司当前盈利的定价倍数。

纳指100 当前整体 PE（TTM）≈ 29.5x
筛选阈值：PE < 行业均值 80%

📈 PEG（市盈增长比）

PE ÷ 盈利增长率。
修正了增长差异，成长股估值的核心指标。

PEG < 1.0 = 低估
筛选阈值：PEG < 1.5

🏢 ROE（净资产收益率）

净利润 ÷ 股东权益。
衡量公司用股东资金赚钱的效率。

巴菲特偏好 ROE > 15%
筛选阈值：连续3年 ROE > 15%

💡 三维交叉验证

单一指标容易误判。PE低可能是盈利不可持续；ROE高可能是高杠杆。三者联用才能找到"好公司 + 好价格"。

课程总结

KEY TAKEAWAYS
📌 本节课核心要点
  • ✅ 漏斗分析 = 逐层筛选思维，AARRR是经典框架
  • ✅ 选股漏斗：全市场 → 指数 → 行业 → 财务 → 估值
  • ✅ 纳指100 = 全球科技龙头集合，年化回报 ~14%
  • ✅ PE + PEG + ROE 三维估值交叉验证
  • ✅ 最终浓缩率 < 0.1%，漏斗让决策有据可依
📝 课后作业
  1. 从纳指100中任选 5 只非科技股，用 ROI 指标替代 ROE 做一次漏斗筛选
  2. 对比 标普500 和 纳指100 的行业分布差异，讨论分散化的必要性
  3. 用 Python 或 Excel 拉取真实数据，复现今天的漏斗筛选流程
  4. 思考：如果加上"股息率 > 1%"这个条件，漏斗出口会怎么变化？
