## [2.8.0] - 2026-07-03 — 账户安全 + 放松训练边界 + 导航状态

### Changed
- 补齐 6 篇睡前故事正文，使估算朗读时长与 UI 显示的「约 4/5 分钟」一致。
- 睡前故事朗读时，故事卡片与底部 MiniPlayer 显示真实播放进度、当前/总时长，并把故事音量与环境声音量分开控制。
- 密码重置后递增会话版本并清除当前 cookie，使旧登录态立即失效；已登录时打开重置链接也会优先显示重置表单。
- `/api/auth/forgot` 不再向浏览器响应重置链接；未配置 SMTP 时仅打印到服务端控制台，避免知道邮箱即可重置密码。
- `lib/server/mailer.ts` 接入真实 SMTP 发送（支持 465 TLS / 587 STARTTLS），配置 `SMTP_*` 后找回密码会发送邮件。
- 未登录时，助眠声音四个分类（自然 / 白噪音 / 环境 / 轻音乐）各保留 1 个可试听声音，其余保留展示但置灰并提示登录后可用。
- 放松训练整体改为登录后可用：呼吸训练、担忧时间、认知重构与冥想入口保留展示但置灰，直接访问训练页也会要求登录。
- 底部 5 Tab 导航选中态增强：当前页增加低刺激底色、描边、阴影与轻微放大，让所处位置更清晰。

### Added
- 新增放松训练访问边界单测；故事时长、会话版本、找回接口不泄露链接、SMTP 投递、助眠声音访问边界等单测继续全绿，合计 **104 条**。

## [2.7.0] - 2026-06-14 — 正念呼吸三档真实时长

### Added
- 新增 `[[pause:N]]` 显式静默停顿标记与 `lib/spoken.ts` 朗读辅助函数，用于把正念练习的真实留白纳入播放时长。
- 新增 MP3 缓存键文本指纹：同一段首次合成后写入 IndexedDB，二次播放直接复用；脚本文本改写后自动避开旧 MP3 缓存。
- 新增 12 条单测：守卫正念 5/10/15 分钟内容互不相同、5 分钟不复用 10 分钟核心引导、估算时长分别接近 302/604/901 秒、静默标记过滤、云端引擎选择、TTS 缓存键文本指纹与缓存命中不请求 API；合计 **95 条全绿**。

### Changed
- 正念呼吸 5 分钟重写为「短暂停顿 / 呼气末端留白」练习，避开 10 分钟的声音扫描、进来/出去标记、担心标签与睡前降速叙事；10/15 分钟保留。
- 含显式静默停顿的脚本在开启 AI 朗读时改为逐段云端播放，并由播放器插入静默，避免把 `[[pause:N]]` 控制标记提交给阿里云或 Edge 云端合成。
- 正念 15 分钟脚本留白从偏长调整到接近标注时长，5/10 分钟补足真实练习留白。

## [2.6.0] - 2026-06-13 — 首页账户入口 + 登录后内容边界

### Added
- 首页账户卡片：登录 / 注册 / 找回 / 重置密码整合到首页；旧 `/login`、`/register`、`/forgot`、`/reset` 路由自动回到首页对应模式。
- 新增登录后功能边界：故事、冥想、阿里云 TTS、Edge TTS 与管理员能力需登录；匿名保留日记、评分、报告、计划、呼吸、助眠声音、CBT-I 工具与本地数据能力。
- 正念呼吸 10 分钟、15 分钟脚本扩写为独立长脚本，不再复用 5 分钟内容。
- 新增单测 1 条，守卫正念 10/15 分钟脚本明显长于 5 分钟版本；合计 **83 条全绿**。

### Changed
- `/api/tts` 增加登录校验，避免匿名直接调用阿里云或 Edge TTS。
- 故事与冥想入口保持可见但置灰，显示「登录后可用」并提供回跳登录。
- 设置页 AI 朗读开关未登录时置灰，登录后才探测服务端 TTS 状态。

## [2.5.0] - 2026-06-08 — 账户体系 + 管理员 + 冥想模块

### Added
- **冥想·放松模块**（spec §4.6 从占位转为已实现）：6 篇原创引导脚本——渐进式肌肉放松（PMR）、身体扫描、正念呼吸 5/10/15 分钟、担忧卸载引导（`lib/meditations.ts`）。
  - 播放页 `app/practice/meditation`：呼吸式光球 + 段落进度 + 暂停/继续，完成记 `PracticeSession`（type=meditation/pmr）。
  - 训练页「冥想 · 放松」分区解除置灰，改为真实可点列表。
- **可选账户体系（原型级服务端）**：注册 / 登录 / 登出 / 密码找回（邮箱重置链接）。
  - 服务端库 `lib/server/{crypto,user-store,session,mailer}.ts`：Node 内置 `crypto`（scrypt 哈希 + HMAC 签名 httpOnly cookie），JSON 文件存储 `.data/users.json`（写互斥锁），SMTP 缺失时回退控制台/页面显示链接。
  - 认证 API `app/api/auth/{register,login,logout,me,forgot,reset}`（重置令牌仅存哈希、30 分钟过期、单次使用、防用户枚举）。
  - 前端 `app/{login,register,forgot,reset}`；store 新增非持久化 `account` 分片；`/me` 加账户入口与登出。
  - 匿名升级：把本地匿名 `userId` 作为 `localUserId` 关联账户，健康数据仍全在本机。
- **管理员控制台**：用户管理（启用停用/删除/重置密码）、内容管理（只读列出故事+冥想）、数据概览（隐私安全聚合 + 注册柱状图）、角色权限。
  - `app/api/admin/{users,users/[id],users/[id]/reset-password,overview,content}` + `app/admin/{layout,page,users,content}`，服务端 `requireAdmin` + 前端路由守卫双重校验。
  - 角色：首个注册用户或 `SEED_ADMIN_EMAIL` 为 admin；防锁死（最后一个管理员不可停用/删除/删自己）。
- 单元测试 +24 条（密码哈希/会话 10、冥想内容 6、TTS 解析 3、用户存储 5），合计 **82 条全绿**。
- 环境变量：`AUTH_SECRET`、`SEED_ADMIN_EMAIL`、`SMTP_*`（见 `.env.example`）。

### Changed
- `lib/tts-engine.ts` 新增 `resolveSpoken()`，统一解析睡前故事与冥想（`med-` 前缀防碰撞），故事播放行为零回归。
- `User` 数据实体新增 `email` / `role` / `disabled` 字段（spec §7.1.1）。
- `.gitignore` 忽略 `.data/`。

### Notes
- 账户 / 管理员 / 云 TTS 需 server 模式；`STATIC_EXPORT` 静态导出与 `hush.html` 单文件版下不可用。生产前必须配置 `AUTH_SECRET`。
- SMTP 真实投递仍是预留位；当前原型回退为控制台/开发页面重置链接。

## [2.0.0] - 2026-05-24 — TTS 引擎定型

### Added
- 阿里云百炼 DashScope CosyVoice v3 作为默认 TTS 引擎（12 音色）。
- 缓存 key 含 engine + voice + speed + text，完全隔离；设置页按引擎动态切换音色列表。

### Changed
- Microsoft Edge TTS 降为手动可选备用引擎。
- 无自动 fallback：引擎失败直接显示错误，便于独立排查。

### Removed
- 彻底移除火山引擎豆包 TTS（APP ID 无法获得资源授权）；清理所有 `VOLC_*` 凭证与相关代码。
