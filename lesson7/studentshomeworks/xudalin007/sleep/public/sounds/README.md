## 选素材建议
- 自然声 ≥ 60s，音乐 2-5 分钟即可（循环播放）
- mp3 192kbps 即可
- 每条 < 10MB
- 避免有突兀声响（电话铃、人声、车辆喇叭等）
- 轻音乐选纯器乐，不要带歌词的人声
